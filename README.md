# 📰 Telegram-бот для сортировки новостей

Telegram-бот для автоматического парсинга, сортировки и отправки актуальных новостей пользователю на основе его интересов.

## 🚀 Возможности
- автоматический парсинг новостей с сайта (Lenta.ru)
- получение актуальных заголовков и ссылок
- отправка пользователю топ-5 новостей
- команда `/update` для обновления базы новостей
- реализован на Python с использованием aiogram
- подготовлена основа для semantic search через embeddings

## 🛠 Технологический стек
- Python 3
- aiogram
- requests
- BeautifulSoup4
- asyncio

## 📌 Текущий MVP
Проект разработан как MVP в рамках тестового задания за ограниченное время.

Реализовано:
- создание и настройка Telegram-бота
- логика парсинга новостей
- отправка топ-5 новостей пользователю
- команда обновления `/update`
- базовая архитектура для дальнейшего масштабирования

Планируемые улучшения:
- подключение базы данных
- хранение интересов пользователей
- система рекомендаций через embeddings
- сортировка по свежести и релевантности
